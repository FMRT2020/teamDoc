# 本部分是关于存储体系部分的设计
## v1.0

### 主要功能

* hanming code （SEC-DED）
* I-Cache：直接映射
* D-Cache：两路组相连
* L2-Cache：两路组相连
* Memory

### 想法

```
初始想法只是在提供一个支持四取指的I-Cache和支持不对齐数据访问的D-Cache
和一个双端口的L2-Cache
对RAM进行SEC-DED编码
```

